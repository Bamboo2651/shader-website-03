# Shader解説: Noise Fade (浸食ディゾルブ)

`slide.frag` 内に記述されている `effectNoiseFade` 関数についての解説です。
単純に薄くなって消える（画像の透明度が変化する）のではなく、「雲」や「砂」のようなノイズ模様に沿って、穴が広がるように燃え尽きていくトランジションです。

## パーリン・シンプレックスノイズ (2D Noise)
自然なしみやモヤモヤとした模様を作り出すのが `noise` 関数です。ランダム関数（ホワイトノイズ）とは違い、グラデーションを持った滑らかな模様が出力されます。
```glsl
float n = noise(uv * 10.0 + vec2(uTime * 0.2, uTime * 0.1));
```
このノイズ値 `n` (0.0 〜 1.0) の模様を使って、しきい値 `p` と比較します。
「ノイズ値が `p` を下回っているピクセルは即座に次の画像に切り替える」という処理（`smoothstep`）を行うことで、モヤモヤしたパターンに沿った切り替わりが実現できます。

## 炎のエッジ表現 (Fringe)
切り替わっている境界線の部分だけに色（炎のようなオレンジなど）をつけると、映像が一気にリッチになります。
```glsl
// 少し広い境界と少し狭い境界を引き算し、「輪郭線」だけを取り出す
float fringe = smoothstep(mappedP - 0.2, mappedP + 0.2, n) - smoothstep(mappedP - 0.05, mappedP + 0.05, n);
vec4 flame = vec4(2.0, 0.8, 0.2, 1.0); // 明るいオレンジ色
```
ここで抽出した境界線(`fringe`)にオレンジ色の光(`flame`)を掛け合わせて、元の画像に足すことで、紙が炎で燃え広がりながら次の画像に変わっていくような演出になっています。
